Multi-Overlay Architecture (Jurojin Style)

Each poker table gets its own tiny RNG overlay. Roll once, ALL overlays update simultaneously.

  • Click ROLL or press Space to generate a number
  • Click + TABLE to add a new overlay (in Mac app: drag it to your table)
  • Auto Roll: set interval, overlays auto-refresh
  • Each overlay is an independent window you can drag to any position
  • Overlays stay on top of all windows including fullscreen poker clients
